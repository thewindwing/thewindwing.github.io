angularjs是什么?

  • 一款非常优秀的前端高级JS框架
  • 由谷歌团队负责开发维护

框架与库的区别

  • 库：对代码进行封装，调用封装的方法，简化操作，如jQuery
  • 框架：提供代码书写规范，按照规则写代码，框架自动帮我们实现相应的功能
  • 无论是angularjs还是jQuery都是用原生JS封装的

指令

  • ng-click=”函数调用” —–为当前元素添加点击事件
    • <button ng-click="val=+val+1"></button>
    • ng-click=”clickfn()”
    • ng-click里面不能写原生JS代码 只能写angularJS中的代码
    • 函数clickfn应该去控制器内去定义
  • ng-model=”变量名”
    • 获取表单元素的值，并且-把获取的值复制给变量
    • 实现数据双向绑定
  • ng-app=“模块名字”
    • 当页面加载完成以后，angularjs会去页面中找ng-app属性
    • 如果没找到，则什么也不做
    • 找到则会执行属性所在元素包裹的代码
    • 可以接受模块名字作为参数
    • angularjs会将当前页面交给指令所指定的模块去管理
  • ng-init=”变量1=值1;变量2=值2”
    • 给变量赋初始值，一定要写在ng-app内部
    • 值若是字符串，要加引号ng-init="我是字符串"
  • ng-view 属性
    • angularjs会将获取到的模版文件内容放在页面中有ng-view指令的元素中

插值表达式 { { } }的作用

  • 在表达式中可以写ng中的变量
  • 显示字符串
  • 进行简单计算 + - * / %
  • 可以写三元运算符
  • 不能写循环和if-else判断

angularjs是基于模块开发的

  • HTML页面和angularjs模块之间是管理和被管理的关系，angularjs是管理者

如何创建模块？

  • 语法：
    • 模块对象=angular.module(‘模块名字’,[要依赖的模块名字])–返回值是模块对象
    • 如果当前模块没有要依赖的模块，第二个参数就要写一个空数组
    • angular.module(’模块的名字’,[] )

如何创建控制器

  • 语法：
    • 模块对象.controller(‘控制器名字’,function($scope){回调函数})
    • 创建多个控制器时，直接复值
    • $scope 专门向控制器控制的区域暴露数据的是一个对象类型
    • 所有的变量都要去在$scope对象下面去找 $scope.变量
    • 兄弟控制器之间如果定义了相同的属性 ，不会冲突
  • 作用：
    • 为用户界面提供数据
    • 用户界面的逻辑主要写在控制器中
  • 告诉angularjs页面中的哪一个区域由哪一个控制器去控制
    • ng-controller=’控制器名字’
    • 告诉angularjs当前区域由这个控制器去控制

angularjs中双向数据绑定

  • 当我们在文本框中写ng-model指令以后，angularjs会监听文本框的值变化事件，当事件发生的时候，angularjs会获取到文本框的最新的值，并且把这个值赋值给$cope下面的相同属性，当$cope下面的属性的值发生变化的时候，angularjs会去页面中找哪一个地方使用了这个变量，将使用到这个变量的地方更新成最新的值，这个现象叫做数据双向绑定。
  • html => js html中通过表单元素将数据改变 js中跟着同步
  • js=>html js中将数据改变 html中也跟着同步
  • 实现双向数据绑定的必要条件
    • 必须是表单元素
    • 且元素身上要有ng-model指令

单页面web应用（single page web application 简称SPA）

  • 特点：
    • 整个网站一个页面构成
    • 链接对应锚点，锚点对应页面
    • 通过ajax局部刷新达到页面切换的目的
    • 公共部分只刷新一次
    • 不会发生页面跳转白屏现象，跳转过程中可以加载loading等
  • 缺点：
    • 不能被搜索以前收入
  • 应用场景
    • 适合不需要被搜索以前搜索到的应用
    • 如网站后台管理系统，办公OA 混合app（在单页面的基础上包了个Android壳子）
  • 监督地址栏中的锚点值发生变化事件
    • window.onhashchange事件
  • 获取锚点值
    • location.hash
  • 判断锚点值
  • 根据锚点值获得对应的页面

在angularjs中实现单页面应用

  • 路由
    • 路由就是用来配置页面之间的跳转关系的，配置锚点与页面之间的对应关系
    • 路由并没有集成在angularjs文件中，而是作为一个单独的模块存在，如果使用路由，需要将路由模块作为主模块的依赖模块
    • 路由模块的名字是ngRoute
    • angular.module('模块名字',['ngRoute'])
    • 模块：
      • 入口模块—管理页面的那个模块
      • 功能模块—路由

路由配置

  • 语法
    • 模块对象.config(回调函数)
    • 回调函数有个参数是路由配置对象$routeProvider
    • 路由配置对象有个when方法由来配置锚点和页面的跳转
    • angularjs要求锚点必须以“/”开头
    • HTML中的锚点要以#!/开头 <a href="#!/index"></a>
    • 路由配置对象有个otherwise方法用来设置默认跳转到首页
      • 页面一打开 地址栏中没有锚点，页面中的坑是空白
      • 当用户输入一个不存在的锚点时

为路由提供模版的三种方法

    1. 外联方式：
    • 使用templateUrl方式载入模板
    • 需要在http环境下(本地file协议下不支持)，发送ajax请求
    • 使用最多

  • 2.内联方式：

    • 使用template载入模版
    • 不发送ajax请求，可以用file或 用localhost

  • 3.内部模版：

    • 通过script标签载入模版
    • 不发送ajax请求 可以用file或 用localhost
