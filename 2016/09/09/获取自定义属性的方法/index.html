获取元素的自定义属性

2016-09-09
  • 获取自定义属性

1. getattribute()方法 （javascript）

<div id='box'></div>
var box=document.getElementById('box');
  • 设置自定义属性 box.setAttribute('data-num','11')
  • 获取自定义属性 box.getAttribute('data-num') // 11
  • 移除自定义属性 box.removeAttribute('data-num') // undefiend
  • 也可以用于系统已定义的属性

2. attr()方法 （jQuery）

  • 设置自定义属性 $('#box').attr('data-num','11')
  • 获取自定义属性 $('#box').attr('data-num') // 11
  • 移除自定义属性 $('#box').removeAttr('data-num') // undefiend
  • 也可以用于系统已定义的属性

3. prop()方法 （jQuery）

  • 设置自定义属性 $('#box').prop('data-num','11')
  • 获取自定义属性 $('#box').prop('data-num') // 11
  • 移除自定义属性 $('#box').removeProp('data-num') // undefiend
  • 注意:多用于系统已定义的属性的获取和设置

4. data()方法 （jQuery）

  • 设置自定义属性 $('#box').data('data-num','11')
  • 获取自定义属性 $('#box').data('data-num') // 11
  • 移除自定义属性 $('#box').removeData('data-num') // undefiend
  • 注意:不可用于系统自带属性

5. dataset方法 （html5）

  • 设置自定义属性 格式：data-属性名
    • 直接在标签内设置 <div data-num-val=22></div>
    • 通过dataset设置 $('#box')[0].dataset['numVal']=22 或 $('#box')[0].dataset.numVal=22
    • 注意：属性名是驼峰命名
  • 获取自定义属性
    • $('#box')[0].dataset['numVal'] 或 $('#box')[0].dataset.numVal
    • 通过jQuery中的data()方法获取html5中自定义的属性$('#box').data('num-val') 直接省去前面的data
  • 移除自定义属性
    • 用上述removeAttribute('data-num-val')、removeAttr('data-num-val')、removeProp('data-num-val')方法均可
    • 手动删除div标签内的自定义属性 <div id='box'></div>
