通过各选择器获取元素大总结

2016-10-15
  • 选择器总结

css中的选择器

1. 基本选择器

  • 通配符选择器 *    *{color:red;}
    • 可选择所有的选择器

  • id选择器 #   #dv{color:red;}

  • 类选择器 .    .box{color:red}

  • 标签选择器    div{color:red;}

2. 关系选择器

  • 后代选择器 (E F) —- 选择所有被E元素包含的F元素（包含子孙）
  • 子代选择器（E>F）—- 选择所有作为E元素的子元素F（不包含孙辈）
  • 交集选择器 (EF) —- 选择E元素，并且E元素具有F类名或者id名 div#dv或div.box
  • 并集选择器(E,F) —- 选择E元素和F元素
  • 相邻选择器(E+F) —- 选择紧贴E元素的F元素，即F元素的上一个兄弟元素是E元素
  • 兄弟选择器(E~F) —- 选择E元素后面的所有兄弟元素F

3. 属性选择器

  • E[attr] —- 选择具有attr属性的E元素
    input[value]{color:red;} //第二个input被选中

  • E[attr=’val’] —- 选择具有attr属性且属性值等于val的E元素 val的引号可有可无

  • E[attr*=val] —- (css3新增) 选择具有attr属性且属性值包含val字符串的E元素
    div[class*=a]{border-color:green} //后三个div都被选中

  • E[attr^=val] —- (css3新增) 选择具有attr属性且属性值为以val开头的字符串的E元素
    div[class^=a]{border-color:plum} //第三个div被选中

  • E[class$=val] —- (css3新增) 选择具有attr属性且属性值为以val结束的字符串的E元素
    div[class$=a]{border-color:pink} //第二个div和第四个div被选中

  • E[attr~=val] —- 选择具有attr属性且属性值仅等于val或者属性值为以空格分隔的字词列表，但其中有一个等于val的E元素
    div[class~=a]{} //第二个和第四个div被选中

  • E[attr|=val] —- 选择具有attr属性且属性值为以val开头并用“-”分隔的字符串的E元素。如果属性值仅为val也被选中
    div[class|=a]{} //第二个和最后一个div被选中

4. 伪类选择器

  • E:link —- 设置链接a在未被访问前的样式
  • E:visited —- 设置链接a在已被访问过的样式
  • E:hover —- 设置元素在鼠标悬停时的样式（不仅是a）
  • E:active —- 设置元素在鼠标点击且不放开始时的样式（不仅是a）
  • E:focus —- 设置元素在输入焦点时的样式（如input获取焦点时）
  • E:first-child —- 匹配E元素且E元素是其父元素的第一个子元素

下面伪类选择器是css3新增

  • E:last-child —- 匹配E元素，且E元素是其父元素的最后一个子元素
  • E:only-child —- 匹配E元素，且E元素是其父元素仅有的一个子元素
  • E:nth-child(n) —- 匹配E元素，且E元素是其父元素的第n个子元素，如果第n个子元素不是E，则无效
    • n的取值范围是0-正无穷大
    • n可以是odd或者even，匹配奇偶数E:nth-child(odd)奇数，E:nth-child(even)偶数也可以E:nth-child(2n+1)奇数,E:nth-child(2n)偶数
    • E:nth-child(-n+3)匹配的是前三个，且都是E元素
    • 注意：n和任何数计算只能放在前面， 5+n 5-n 都不对
  • E:nth-last-child(n) —- (css3) 匹配E元素是其父元素的倒数第n个子元素，如果倒数第n个子元素不是E，则无效
  • E:first-of-type —- 匹配元素E，且E元素是同类兄弟元素的第一个元素
  • E:last-of-type —- 匹配元素E，且E元素是同类兄弟元素的最后一个元素
  • E:only-of-type —- 匹配元素E，且E元素是同类兄弟元素的仅有的一个，即没有同类别的兄弟元素
  • E:nth-of-type(n) —- 匹配元素E，且E元素是同类兄弟元素的第n个元素
  • E:nth-last-of-type(n) —- 匹配元素E，且E元素是同类兄弟元素的倒数第n个元素
    • E:not(selector) —- 匹配不含有选择器selector的元素E
  • E:empty —- 匹配E元素，且E元素没有任何子元素包括text节点
  • E:checked —- 匹配input 的type为radio和CheckBox时，处于选中状态的元素E
  • E:enabled —- 匹配处于用以状态的E元素 input
  • E:disabled —- 匹配处于禁用状态的E元素 主要用于input textarea
  • E:target —- a的URL是锚点#，即E元素是a元素的目标元素，当点击a时，目标元素E改变类样式

5. 伪元素（对象）选择器

  • E:first-letter/E::first-letter —-（CSS1/3）设置元素E的第一个字符的样式（第一个汉字或字母）
  • E:first-line/E::first-line —-（CSS1/3） 设置元素E的第一行的样式
  • E:after/E::after —-（CSS2/3）在元素E的内部结束位置创建一个行内元素，必须结合content一起使用a::after{content:'链接' }
  • E:before/E::before —（CSS2/3）在元素E的内部开始位置创建一个行内元素，必须结合content一起使用
  • E::selection —-（css3）设置元素E中的内容被选择时的颜色（背景色，字体颜色）

jQuery中的选择器

1.基本选择器和关系选择器

同上面css的基本选择器一样,除了这些外，jQuery还有很多基本的选择器：

  • :first
    • $(“p:first”) 获取第一个p元素
    • 同 对象.first()方法
    • 同css的:first-of-type
  • :last
    • $(“p:last”) 获取最后一个p元素
    • 同 对象.last()方法
    • 同css的:last-of-type
  • :even
    • 匹配所有索引值为偶数的元素，从 0 开始计数
    • 同css的:nth-child(odd) $("tr:even"),页面上显示的为奇数行
  • :odd
    • 匹配所有索引值为奇数的元素，从 0 开始计数 $("tr:odd")
  • :not(selector)
    • 去除所有与给定选择器匹配的元素
  • :focus
    • 匹配当前获取焦点的元素
    • 同css的：focus
  • :header
    • 匹配如 h1, h2, h3之类的标题元素

2.属性选择器

  • 与css选择器的不同是，没有E[attr~=val]和E[attr|=val]两个选择器。
  • 但多了一个复合属性选择器：[selector1][selector2][selectorN] 当某个元素满足多个条件时使用,如$("div[id][name$='lily']")

3.子元素选择器

  • :first-child
  • :first-of-type
  • :last-child
  • :last-of-type
  • :nth-child(n)
  • :nth-last-child(n)
  • :nth-last-of-type(n)
  • :nth-of-type(n)
  • :only-child
  • :only-of-type
  • 以上同css的相应选择器

4.索引选择器

  • :eq(index) —-匹配索引值为index的元素 index从0开始计数
  • :lt(index) —-匹配所有小于索引值index的元素
  • :gt(index) —-匹配所有大于索引值index的元素
  • 对象.get(index)方法 —-取得第index个位置上的元素
  • 对象.index(selector|element)方法 —-搜索匹配的元素，并返回元素的索引值。从0开始

4.表单选择器

  • :input —- $(':input') 匹配所有input、select、textarea、button元素
  • :text —- $(':text') 匹配所有type=text的单行文本框
  • :password —- $(':password') 匹配所有type=password的密码框
  • :radio — $(':radio') 匹配所有单选按钮
  • :checkbox —- $(':checkbox') 匹配所有复选框
  • :submit —- $(':submit') 匹配所有提交按钮
  • :image —- $(':image') 匹配所有type=image的图像区域
  • :reset —- $(':reset') 匹配所有重置按钮
  • :button —- $(':button') 匹配所有按钮，包括type=button或者button标签
  • :file —- $(':file') 匹配所有的文件域即type=file
  • :enabled —- $('input:enabled') 匹配所有的可用元素
  • :disabled —- $('input:disabled') 匹配所有不可用元素
  • :checked —- $('input:checked') 匹配所有被选中的元素，包括单选框和复选框
  • :selected —- $('select option:selected') 匹配所有选中的option元素

6.内容选择器

  • :contains(text) —- 匹配包含指定文本text的元素
  • :empty —- 匹配不含有子元素或文本为空的元素
  • :has(selector) —- 匹配所有含有选择器所匹配元素的元素
  • ：parent —- 匹配含有子元素或者文本的元素

7.可见性

  • :hidden —- 匹配所有不可见元素或者type为hidden的元素
  • :visible —- 匹配所有的可见元素
