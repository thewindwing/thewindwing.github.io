正则表达式

2017-01-10
  • 正则表达式

常用的基本元字符

.   表示除了\n换行符以为的任意一个字符

[ ]   表示一个区域

  • [a-z]表示a-z之间的任意一个即所有小写字母中的任意一个
  • [A-Z] 表示大写字母中的任意一个
  • [a-zA-Z]表示字母中的任意一个
  • [0-9]表示任意一个数字

{n}    表示前面的字符刚刚发生了n次

*   表示前面的表达式出现了0次货多次{0，}

+   表示前面的表达式出现了1次到多次{1，}

？   表示前面的表达式出现了0次到1次{0，1}

\d   表示任意的数字 [0-9]

\D   表示任意的非数字 [^0-9]

\s   任意的空白字符

\S   任意的非空白字符

\w   任意的非特殊字符，包括下划线‘’ [0-9a-zA-Z]

\W    任意的特殊字符， [^0-9a-zA-Z_]

^   以什么开始，取反

$   以什么结束

定义正则表达式

  • 方法一： var reg=/元字符集/
  • 方法二： var reg=new RegExp(/元字符集/);

常用的RegExp对象的方法

1.  reg.test(string)

检测字符串string是否匹配reg表达式，返回true或false

2.  reg.exec(string)

检测字符串中的正则表达式匹配，返回一个数组，其中存放的是匹配的结果，匹配不到返回null

支持正则表达式的String的对象的方法

1.  string.match(reg)

在string字符串中找到一个或多个reg正则的匹配值，返回一个数组，未匹配到时返回null

  • 若reg没在全局下查找，则match只执行一次，返回一个数组
  • 若reg在全局下查找，则返回的数组式匹配到的所有字符

2.  string.split(reg)

把字符串string按正则匹配的值进行分隔成数组

console.log(str.split(reg1)); // ["v", "ni", "r", "=%"]

3.  string.search(reg)

检测字符串中与reg正则匹配的子串，找到则返回首次出现的起始位置，否则返回-1

console.log(str.search(reg1)); // 1

4.  string.replace(reg,替换的文本)

在字符串中查找reg匹配的子串，并替换为要替换的文本，返回结果是替换后的新字符串

console.log(str.replace(reg1,0)); // v0ni0r0=%
