函数查找机制

所有函数的原型对象都指向Function.prototype，包括Function自己

f1.__proto__===Function.prototype;
Function.__proto__===Function.prototype;

函数的自定义方式

2015-06-08
  • 函数的自定义方式
  • 函数

函数声明

function f1(){}

  • 最大特性：预解析时具有函数提升
  • 实例：
    console.log(typeof f1);//这里输出的是function，不是number
    var f1=123;
    function f1(){
      console.log(‘hello’);
    }
    console.log(f1); //这里的结果是123
    • 解析：
  • 因为预解析时，同名函数和变量，函数的优先级高，与先后顺序无关。
  • 在执行阶段，因为为函f1重新复制了123，而并没调改函数f1，所以会直接跳过function f1(){}执行下面的代码。

函数表达式

var f1=function(){}

  • 类似于变量赋值，没有函数提升
  • 可以没有名字（匿名函数）

new Function()创建函数

var sum=new Function(‘x’,’y’,’return x+y’);
和下面的sum函数一样
function sum(x,y){
  return x+y;
}
