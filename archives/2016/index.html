同源和跨域请求

2016-11-11
  • 同源和跨域请求

同源请求

同源策略是浏览器的一种安全策略，为了保证浏览器的安全性，服务器不允许其他来源的请求，只允许自己网站的请求。
协议名、域名和端口都一样的网站是同源网站

跨域请求

只要协议名、域名或端口名有一个不一样就是跨域，跨域的网站只允许浏览其他页面但不能操作这个页面。

  • 具有跨域能力的标签
    1. iframe标签
    2. link标签
    3. script标签
    4. img标签
    5. audio标签
    6. video标签
  • 跨域解决方案
    • domain+iframe
      顶级域名相同的情况下可以通过同时设置document.domain=顶级域名就可以互相操作了
    • hash+iframe
      • name+iframe
      • postMessage+iframe 只能解决跨域的数据交互问题
    • 通过jsonp解决不同域名之间的数据交互
      jsonp的本质是通过标签的可跨越性，xhr.open()只能解决同源之间的数据交互问题，并不能实现跨域之间的数据交互，有些标签却具有跨域请求的能力，我们可以利用这些标签实现跨域交互数据，由服务端返回一个预先定义好的callback函数的调用，并将服务器数据通过该函数的参数的形式传递过来。

通过各选择器获取元素大总结

2016-10-15
  • 选择器总结

css中的选择器

1. 基本选择器

  • 通配符选择器 *    *{color:red;}
    • 可选择所有的选择器

  • id选择器 #   #dv{color:red;}

  • 类选择器 .    .box{color:red}

  • 标签选择器    div{color:red;}

2. 关系选择器

  • 后代选择器 (E F) —- 选择所有被E元素包含的F元素（包含子孙）
  • 子代选择器（E>F）—- 选择所有作为E元素的子元素F（不包含孙辈）
  • 交集选择器 (EF) —- 选择E元素，并且E元素具有F类名或者id名 div#dv或div.box
  • 并集选择器(E,F) —- 选择E元素和F元素
  • 相邻选择器(E+F) —- 选择紧贴E元素的F元素，即F元素的上一个兄弟元素是E元素
  • 兄弟选择器(E~F) —- 选择E元素后面的所有兄弟元素F

获取元素位置的属性和方法

2016-10-14
  • 获取元素位置

DOM中的offset系列

  • element.offsetLeft —- 获取的是元素相对于可视区左边的距离，当父元素定位后，则相对于父元素
  • element.offsetTop —- 获取的是元素相对于可视区顶部的距离，当父元素定位后，则相对于父元素
  • element.offsetWidth —- 获取的是元素的宽度，包括border，即=width + padding + border。并不能通过element.width来获取style标签中定义的元素的宽度，可以用这个方法来直接获取
  • element.offsetHeight —- 获取的是元素的高度
  • e.offsetX —- 获取的是发生事件的地点在事件源的坐标系中的X坐标
  • e.offsetY —- 获取的是发生事件的地点在事件源的坐标系中的Y坐标

git

2016-05-09
  • git

1. shell

  • shell是一个命令解析器，它接收用户命令，然后调用相应的应用程序
  • shell是Linux中的命令解析器，dos是Windows中的命令解析器如cmd命令窗口
  • 分类：csh ksh bash(Bourne Again shell)等，bash用的最多。

2. bash常见命令

  • 格式 — 命令 [选项] [参数],如 ls -a ./abc
  • pwd 查看当前目录
  • cd 目录名 — 切换目录
  • ls 查看当前目录下的内容
    • ls -a
    • ls -a -s
    • ls -al
  • mkdir 目录名 — 创建目录
  • touch 文件名 — 创建文件
  • cat 文件名 — 查看文件的全部内容
    • tail查看文件的尾部 cat 文件名 tail
  • less 文件名 — 查看文件，有折叠
    • 每次按一次回车多出现一行
    • 按b回退一页
    • 按 f或空格向前翻一页
    • q退出
  • rmdir 空文件夹名 — 删除文件夹，只能删除空文件
  • rm 文件名 — 删除文件
  • rm -rf 目录名 — 删除文件夹
  • mv — 移动文件或重命名
    • mv 要移动的文件名 位置
    • mv 要重命名的文件 新名字
  • cp 文件 位置 — 复制文件
  • tab — 自动补全，连续按两次会将所有的匹配内容全部显示出来
  • >和>> — 重定向
    • >覆盖 echo hello > apache.html
    • >>追加到尾部 echo ‘hello’ >> apache.html
  • | — 管道符，可以将多个命令连接使用，上一次命令的执行结果当成下一次命令的参数
    • grep 匹配内容，一般结合管道符使用
    • ls | grep ‘h’ 匹配h是否在ls结果中
