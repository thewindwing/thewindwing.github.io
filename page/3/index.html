git

2016-05-09
  • git

1. shell

  • shell是一个命令解析器，它接收用户命令，然后调用相应的应用程序
  • shell是Linux中的命令解析器，dos是Windows中的命令解析器如cmd命令窗口
  • 分类：csh ksh bash(Bourne Again shell)等，bash用的最多。

2. bash常见命令

  • 格式 — 命令 [选项] [参数],如 ls -a ./abc
  • pwd 查看当前目录
  • cd 目录名 — 切换目录
  • ls 查看当前目录下的内容
    • ls -a
    • ls -a -s
    • ls -al
  • mkdir 目录名 — 创建目录
  • touch 文件名 — 创建文件
  • cat 文件名 — 查看文件的全部内容
    • tail查看文件的尾部 cat 文件名 tail
  • less 文件名 — 查看文件，有折叠
    • 每次按一次回车多出现一行
    • 按b回退一页
    • 按 f或空格向前翻一页
    • q退出
  • rmdir 空文件夹名 — 删除文件夹，只能删除空文件
  • rm 文件名 — 删除文件
  • rm -rf 目录名 — 删除文件夹
  • mv — 移动文件或重命名
    • mv 要移动的文件名 位置
    • mv 要重命名的文件 新名字
  • cp 文件 位置 — 复制文件
  • tab — 自动补全，连续按两次会将所有的匹配内容全部显示出来
  • >和>> — 重定向
    • >覆盖 echo hello > apache.html
    • >>追加到尾部 echo ‘hello’ >> apache.html
  • | — 管道符，可以将多个命令连接使用，上一次命令的执行结果当成下一次命令的参数
    • grep 匹配内容，一般结合管道符使用
    • ls | grep ‘h’ 匹配h是否在ls结果中

浅拷贝
