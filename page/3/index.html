angularjs是什么?

  • 一款非常优秀的前端高级JS框架
  • 由谷歌团队负责开发维护

框架与库的区别

  • 库：对代码进行封装，调用封装的方法，简化操作，如jQuery
  • 框架：提供代码书写规范，按照规则写代码，框架自动帮我们实现相应的功能
  • 无论是angularjs还是jQuery都是用原生JS封装的

1. shell

  • shell是一个命令解析器，它接收用户命令，然后调用相应的应用程序
  • shell是Linux中的命令解析器，dos是Windows中的命令解析器如cmd命令窗口
  • 分类：csh ksh bash(Bourne Again shell)等，bash用的最多。

2. bash常见命令

  • 格式 — 命令 [选项] [参数],如 ls -a ./abc
  • pwd 查看当前目录
  • cd 目录名 — 切换目录
  • ls 查看当前目录下的内容
    • ls -a
    • ls -a -s
    • ls -al
